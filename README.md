## 📌 Overview

This project presents an intelligent **Medical AI system** that integrates **Deep Learning** and **Natural Language Processing (NLP)** to assist users in:

* Diagnosing diseases from medical images
* Interacting with an AI chatbot
* Receiving responses in the **user’s native language automatically**

The system is designed to improve **accessible, real-time healthcare assistance**, especially for diverse linguistic users.

---

## 🚀 Key Features

✨ **Chest X-ray Analysis**

* Detects **Pneumonia** from uploaded X-ray images

✨ **Skin Cancer Classification**

* Classifies skin lesions as **Benign** or **Malignant**

✨ **AI Chatbot Assistance**

* Answers general medical queries
* Provides basic healthcare guidance

✨ **Automatic Language Detection 🌍**

* Detects user input language
* Responds in the same language (no manual selection required)

✨ **Real-time Prediction System**

* Fast and efficient diagnosis output

## 🏗️ System Workflow

1. User uploads a medical image (X-ray or skin lesion)
2. Image preprocessing is applied (resize, normalization)
3. Appropriate deep learning model is selected
4. Model generates prediction (disease classification)
5. User submits a query through chatbot
6. System detects the **input language automatically**
7. Chatbot generates a response in the **same detected language**

---

## 📸 Output Highlights

### 🩻 Chest X-ray Diagnosis

* Upload a chest X-ray image
* System analyzes lung patterns using CNN
* **Output:**

  * ✔ Pneumonia Detected
  * ✔ Normal (No Infection)

---

### 🧴 Skin Cancer Detection

* Upload a skin lesion image
* Model evaluates features such as texture, color, and irregularity
* **Output:**

  * ✔ Benign (Non-cancerous)
  * ✔ Malignant (Potential Cancer)

---

### 🤖 Chatbot Interaction

* User asks health-related questions
* Bot provides context-aware responses
* **Example:**

  * *User:* "What are pneumonia symptoms?"
  * *Bot:* "Common symptoms include cough, fever, and difficulty breathing."

---

### 🌍 Intelligent Multilingual Response

* No manual language selection needed

* System detects user language automatically

* **Example:**

  * *User (Tamil):* "நிமோனியா அறிகுறிகள் என்ன?"
  * *Bot (Tamil):* "இருமல், காய்ச்சல் மற்றும் மூச்சுத் திணறல் முக்கிய அறிகுறிகள்."

---

### ⚡ Real-Time Output
